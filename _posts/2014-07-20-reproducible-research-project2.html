Synopsis

This report analyzes the NOAA storm data and finds out that tornado, heat and floods are the three most harmful events to population health (measured by number of fatalities and injuries). With respect to economic damages, floods have done the biggest damage since 1950 to 2011.

Questions we need to answer within this analysis are:

  1. Across the United States, which types of events (as indicated in the EVTYPE variable) are most harmful with respect to population health?
  2. Across the United States, which types of events have the greatest economic consequences?

To answer these questions, we choose the following variables to do the analysis.

  • EVTYPE: Demonstrates the type of the storm;
  • FATALITIES: Number of deaths caused by the storm event;
  • INJURIES: Number of injuries caused by the storm event;
  • PROPDMG: Measures the damage of properties;
  • PROPDMGEXP: An multiplier for PROPDMG, where where Hundred (H), Thousand (K), Million (M), Billion (B) and numbers stand for the power of ten, i.e., 5 means that the multiplier is 105
  • CROPDMG: Measures the damage of crops;
  • CROPDMGEXP: An multiplier for CROPDMG, where where Hundred (H), Thousand (K), Million (M), Billion (B) and numbers stand for the power of ten, i.e., 5 means that the multiplier is 105

Now save all these variables within a new object named newdata. 

newdata <- data.frame(EVTYPE=data$EVTYPE, FATALITIES=data$FATALITIES, INJURIES=data$INJURIES, PROPDMG=data$PROPDMG, PROPDMGEXP=data$PROPDMGEXP, CROPDMG=data$CROPDMG, CROPDMGEXP=data$CROPDMGEXP);

Then using the event table, we can categorize the EVTYPE. If one observation, if the value contains any keyword in the event table, we will count that observation as in that category. If the value doesn't contain any keyword from the table, we just leave it there as it is. Using the grep() function we created a new column, called NEWEVENT, in the new data set. Codes are shown as below.

i <- 1;
w <- as.character(newdata$EVTYPE);
for(cat in eventtype$Keyword){
  index <- grep(toupper(cat), newdata$EVTYPE, ignore.case = TRUE);
  w[index] <- as.character(eventtype$Event[i]);
  i <- i + 1;
}
newdata <- data.frame(NEWEVENT = w, newdata);

Thus we now have a more consistent column for the event type variable. Next, we need to fixed some missing or inconsistent data of other variables. Codes are shown as below.

# Replace all missing values with 0
newdata$FATALITIES[(newdata$FATALITIES == "")] <- 0;
newdata$INJURIES[(newdata$INJURIES == "")] <- 0;
newdata$PROPDMG[(newdata$PROPDMG == "")] <- 0;
newdata$CROPDMG[(newdata$CROPDMG == "")] <- 0;

# Only use numbers for variables of PROPDMGEXP and CROPDMGEXP
newdata$PROPDMGEXP <- as.character(newdata$PROPDMGEXP);
newdata$CROPDMGEXP <- as.character(newdata$CROPDMGEXP);

newdata$PROPDMGEXP[(newdata$PROPDMGEXP == "+") | (newdata$PROPDMGEXP == "-") | (newdata$PROPDMGEXP == "?") | (newdata$PROPDMGEXP == "")] <- 0;
newdata$PROPDMGEXP[(newdata$PROPDMGEXP == "h") | (newdata$PROPDMGEXP == "H")] <- 2;
newdata$PROPDMGEXP[(newdata$PROPDMGEXP == "k") | (newdata$PROPDMGEXP == "K")] <- 3;
newdata$PROPDMGEXP[(newdata$PROPDMGEXP == "m") | (newdata$PROPDMGEXP == "M")] <- 6;
newdata$PROPDMGEXP[(newdata$PROPDMGEXP == "B")] <- 9;

newdata$CROPDMGEXP[(newdata$CROPDMGEXP == "+") | (newdata$CROPDMGEXP == "-") | (newdata$CROPDMGEXP == "?") | (newdata$CROPDMGEXP == "")] <- 0;
newdata$CROPDMGEXP[(newdata$CROPDMGEXP == "h") | (newdata$CROPDMGEXP == "H")] <- 2;
newdata$CROPDMGEXP[(newdata$CROPDMGEXP == "k") | (newdata$CROPDMGEXP == "K")] <- 3;
newdata$CROPDMGEXP[(newdata$CROPDMGEXP == "m") | (newdata$CROPDMGEXP == "M")] <- 6;
newdata$CROPDMGEXP[(newdata$CROPDMGEXP == "B")] <- 9;

newdata$PROPDMGEXP <- as.numeric(newdata$PROPDMGEXP);

Now the data set, newdata, is ready for doing analysis.

Analysis

Using the new data set we just created, in order to evaluate the damage to population health and economics, we need to calculate the total damage for each event. 

econdamage <- newdata$PROPDMG * (10 ^ newdata$PROPDMGEXP) + newdata$CROPDMG * (10 ^ newdata$CROPDMGEXP);
newdata <- data.frame(newdata, ECONDAMAGE = econdamage);

# Calculate the total damage of each type of events.
totalecondamage <- sapply(split(newdata$ECONDAMAGE, newdata$NEWEVENT), sum);
totalecondamage <- totalecondamage[order(totalecondamage, decreasing=TRUE)];
totalfataldamage <- sapply(split(newdata$FATALITIES, newdata$NEWEVENT), sum);
totalfataldamage <- totalfataldamage[order(totalfataldamage, decreasing=TRUE)];
totalinjuredamage <- sapply(split(newdata$INJURIES, newdata$NEWEVENT), sum);
totalinjuredamage <- totalinjuredamage[order(totalinjuredamage, decreasing=TRUE)];
